# 제 평가는요? (MyRatingIs)

**흑백요리사 컨셉의 프로젝트/MVP 평가 솔루션**

창작자들의 설익은 아이디어를 전문가들이 냉철하게 평가하고 성장을 돕는 스테이지입니다.

---

## ✨ 주요 기능

### 👨‍🍳 **셰프 컨셉의 UI/UX**

- 흑백요리사(Culinary Class Wars) 테마의 다크 모드 디자인
- 프로젝트를 '요리'로, 평가를 '심사'로 비유한 인터랙스
- 프리미엄한 블랙 & 오렌지 포인트 컬러 시스템

### 📊 **심도 있는 프로젝트 평가**

- ✅ 프로젝트 업로드 (이미지, 제목, 설명, 카테고리)
- ✅ 6가지 핵심 지표 평가 (독창성, 시장성, 완성도, 기술력, 디자인, 성장성)
- ✅ 전문가 및 게스트 피드백 시스템
- ✅ 실시간 평가 리포트 및 지수(Index) 생성

### 👤 **인증 및 계정 시스템**

- ✅ 이메일 6자리 인증 및 가입
- ✅ 구글 소셜 로그인 연동 (Industry Standard SSR 구현)
- ✅ 이메일 인증 메일 재전송 기능
- ✅ 마이페이지 기반 활동 내역 관리

---

## 🛠️ 기술 스택

- **프레임워크**: Next.js 14.1.3 (App Router)
- **언어**: TypeScript 5
- **스타일링**: Tailwind CSS
- **인프라/백엔드**: Supabase (Auth, DB, Storage)
- **이메일**: Supabase Auth + Resend (준비 중)
- **AI**: Google Gemini Pro (프로젝트 분석 및 요약)

## 📦 설치 및 실행

### 1. 저장소 클론

```bash
git clone <repository-url>
cd myratingis
```

### 2. 의존성 설치

```bash
npm install
```

### 3. 개발 서버 실행

```bash
npm run dev
```
