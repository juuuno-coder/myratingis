# 프로젝트 서비스 정의서: "제 평가는요?" (MyRatingIs)

본 문서는 "흑백요리사" 컨셉의 프로젝트/MVP 평가 솔루션 **"제 평가는요?" (MyRatingIs)**의 기능 개발 및 구현 계획을 정의합니다.

---

## 1. 개요

- **서비스명**: 제 평가는요? (MyRatingIs)
- **컨셉**: 흑백요리사(Culinary Class Wars) 요리 평가 시스템을 모티브로 한 냉철하고 감각적인 프로젝트 평가 플랫폼.
- **주요 가치**: 누구나 자신의 결과물을 전문가나 대중에게 '미슐랭 스타일'로 평가받고, 인사이트를 얻을 수 있는 환경 제공.

---

## 2. 핵심 기능 상세

### 2.1. 평가 의뢰하기 (Create Project)

평가 대상과 평가 기준을 설정하는 단계입니다.

- **미디어 설정**:
  - 웹 링크(URL) 첨부 또는 이미지/스크린샷 업로드.
- **평가 항목 설정**:
  - **Type A. 미슐랭 평가 (Radar Chart)**:
    - 5.0 만점 척도.
    - 최소 3개 ~ 최대 6개 항목 선택 (기본 4개 제공).
    - 프리셋: 기획력, 독창성, 심미성, 완성도, 상업성 등.
  - **Type B. 스티커 투표 (Selection)**:
    - 아이콘/이미지 + 명칭 + 가이드라인 조합.
    - 최소 2개 ~ 최대 6개 (기본 3개).
    - 프리셋 3종 제공:
      1. [몰입형] 현업 전문가 톤
      2. [미슐랭형] 미식 가이드 톤
      3. [MZ·위트형] 직관적 반응 톤
  - **Type C. 종합 의견 (Subjective)**:
    - 장문 주관식 질문 설정.
    - 최소 1개 ~ 최대 3개 (기본 1개).
- **정보 입력**: 프로젝트 명, 한 줄 설명, 평가 완료 기간(D-Day).

### 2.2. 평가하기 (Evaluation Experience)

평가자가 프로젝트를 경험하고 점수를 매기는 단계입니다.

- **인트로 (Intro)**:
  - 흑백요리사 컨셉의 블랙 스킨 배경.
  - '요리 커버(Cloche)' 애니메이션 및 오픈 효과음 적용.
  - '평가 시작하기' 버튼으로 진입.
- **평가 환경 (Viewer)**:
  - **PC**: 좌측 프로젝트 미리보기(Iframe/Image) + 우측 평가 폼 (450px 고정) + 중앙 리사이즈 바.
  - **모바일**: 상단 미리보기 + 하단 평가 폼 (Bottom Sheet 또는 스크롤 구조).
  - 유틸리티: PC/Mobile 뷰 전환, 새창 열기 (인트로 이후 노출).

### 2.3. 결과 확인 및 리포트 (Report)

누적된 평가 데이터를 시각화하여 제공합니다.

- **데이터 저장**: Supabase DB 연동.
- **리포트 구성**:
  - 미슐랭 항목별 평균 점수 (방사형 차트).
  - 스티커 득표 현황 (막대 차트 또는 리스트).
  - 종합 의견 리스트.
- **개인화 옵션**: 공유 여부 설정 (공개/비공개).
- **바이럴 기능**:
  - 결과 요약 이미지 생성 (인스타그램 스토리 최적화 레이아웃).
  - SNS 공유 (카카오톡, 텔레그램 OG 태그 최적화).

---

## 3. 기술 스택 및 라이브러리 (Stack)

- **Framework**: Next.js 14+ (App Router), React.
- **Styling**: Tailwind CSS, Aceternity UI (감각적인 애니메이션 및 컴포넌트).
- **Database/Auth**: Supabase (PostgreSQL, Auth, Storage).
- **Icons**: Font Awesome 6.
- **Others**: Recharts (데이터 시각화), Lucide React/Framer Motion.

---

## 4. 디자인 시스템 가이드

- **Main Color**: Black & Gold (흑백요리사 미슐랭 포인트).
- **Typography**: 프리미엄 고딕/명조체 (Pretendard 등).
- **Aesthetics**: Glassmorphism, Dark Mode, Micro-interactions.
